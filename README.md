# inverse-kinematics

This is a simulation of a simple 3 degree of freedom arm where the third joint controls the end-effector orientation. It maintains alignment with the axes of a Cartesian coordinate system.

<img src="/assets/inverse-kinematic.gif" alt="Inverse kinematics demo" width="360" height="360"/>

# Installation
1. Install [Processing](https://processing.org/download)
2. Run ``IKinematics.pde``

# Controls
**Mouse**: Drag the mouse to move the robotic arm.
